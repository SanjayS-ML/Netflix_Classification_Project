# Netflix Titles Classification Project

## Description
This project classifies Netflix titles into categories using supervised Machine Learning techniques. The project covers data preprocessing, model training, evaluation, and visualization of results.

## Project Steps
1. Data Loading and Exploration
2. Data Preprocessing
   - Handling missing values
   - Encoding categorical features
   - Scaling numerical features
3. Model Training
   - Logistic Regression
   - Decision Tree
   - Random Forest
4. Model Evaluation
   - Accuracy, Precision, Recall, F1-Score
   - Confusion Matrix
5. Visualization of Results
   - Feature importance
   - Comparison of model performance

## How to Run
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Open `Netflix_Classification.ipynb` in Jupyter Notebook
